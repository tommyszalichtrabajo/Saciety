# Saciety 🍽️

Recipe app that suggests what you can cook based on your available ingredients.

## Features

- User authentication
- Ingredient-based recipe suggestions
- Recipe recommendations with up to 2 missing ingredients
- Save favorite recipes
- Create and share your own recipes

## Tech Stack

### Frontend

- React + TypeScript
- Tailwind CSS
- React Query
- Zustand
- React Hook Form

### Backend

- Node.js + Express
- JWT Authentication
- Stripe
- Zod
- Nodemailer

### Database

- MySQL + Prisma
- Redis
